迭代与递归

Posted on Dec 24, 2024

递归与迭代的区别

  • 递归：函数在其定义中直接或间接调用自身。递归通常包含一个或多个基本情况，以防止无限调用。

  • 迭代：通过循环结构（如 for 或 while 循环）重复执行代码，直到满足某个条件为止。

以下是用 Python 实现阶乘的递归和迭代版本。

递归实现

def factorial_recursive(n):
    if n < 0:
        raise ValueError("负数没有阶乘")
    elif n == 0:
        return 1
    else:
        return n * factorial_recursive(n - 1)

# 测试递归函数
print(factorial_recursive(5))  # 输出 120

迭代实现

def factorial_iterative(n):
    if n < 0:
        raise ValueError("负数没有阶乘")
    result = 1
    for i in range(1, n + 1):
        result *= i
    return result

# 测试迭代函数
print(factorial_iterative(5))  # 输出 120

小结

  • 递归实现通常更简洁、易于理解，但在处理较大输入时可能导致栈溢出。
  • 迭代实现在性能上更高效，特别是在输入较大时，但相对而言代码会略显复杂。
